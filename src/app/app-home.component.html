Cloud is the future of hosting whether you like it or not and trust it or not! While Azure and AWS made the cloud adoption easier, they are continuously moving the goal posts to serve their clients better. For us, Developers/Evangelists, the possibilities are unlimited but things can get spoiled quickly unless we adhere to best practices. As I start the journey to cloud, I would be building proof-of-concept applications and deploying them to Azure following the best practices. What's new in this release? OAUTH2 Authentication with LinkedIn, Azure Tables for Idenity Storage, Live Demo of oAuth login, Jwt Bearer Token Authentication for protected API , End-to-End Security with HTTPS/TLS and Blogs to show recent posts from WordPress Blogs site.

While I could have built this portal with ASP.Net Core/MVC, I decided to use Angular 5. This application is built with angular-cli and deployed to IIS. The address list displayed above is for demo. Don't try to dial any of the phone number! You can click an address item to see and update the details as you wish!!.
